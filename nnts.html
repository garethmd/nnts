nnts.metrics.abs_error(y_hat: tensor, y: tensor, dim=1) → tensor

Calculates the absolute error between the predicted values and the actual values.

The function computes the absolute error by taking the element-wise absolute difference between the predicted values and the actual values, and then summing these differences. If the input tensors have more than one dimension, the summation is done along dimension 1.

Parameters:

  • y_hat (torch.tensor) – The predicted values.

  • y (torch.tensor) – The actual values.

Returns:

A tensor containing the sum of absolute errors.

Return type:

torch.tensor

Example

>>> y_hat = torch.tensor([2.5, 0.0, 2.0, 8.0])
>>> y = torch.tensor([3.0, -0.5, 2.0, 7.0])
>>> error = abs_error(y_hat, y)
>>> print(error)
tensor(2.0)

nnts.metrics.msmape(y_hat: tensor, y: tensor, dim: int = 1, eps=0.1) → tensor

Calculates the Modified Symmetric Mean Absolute Percentage Error (msMAPE) between the predicted values and the actual values.

The function computes the msMAPE by calculating the modified symmetric absolute percentage error for each prediction, which is given by the formula 2 * |y_hat - y| / max((|y| + |y_hat| + eps), (0.5 + eps)). If the input tensors have more than one dimension, the mean is calculated along the specified dimension.

Parameters:

  • y_hat (torch.tensor)) – The predicted values.

  • y (torch.tensor)) – The actual values.

  • dim (int, optional) – The dimension along which to compute the mean if the input tensors have more than one dimension. Defaults to 1.

  • eps (float, optional) – Epsilon value to prevent division by zero. Defaults to 0.1.

Returns:

A tensor containing the modified symmetric mean absolute percentage error.

Return type:

torch.tensor

Example

>>> y_hat = torch.tensor([[1.0, 2.0, 3.0, 4.0], [1.0, 2.0, 3.0, 4.0]])
>>> y = torch.tensor([[0.0, 4.0, 6.0, 8.0], [0.0, 2.0, 3.0, 4.0]])
>>> error = msmape(y_hat, y)
>>> print(error)
tensor([1.0, 0.5])

Calculates the Symmetric Mean Absolute Percentage Error (sMAPE) between the predicted values and the actual values.

The function computes the sMAPE by calculating the symmetric absolute percentage error for each prediction, which is given by the formula 2 * |y_hat - y| / (|y| + |y_hat|). If the input tensors have more than one dimension, the mean is calculated along the specified dimension.

  • y_hat (torch.tensor) – The predicted values.

  • y (torch.tensor) – The actual values.

  • dim (int, optional) – The dimension along which to compute the mean if the input tensors have more than one dimension. Defaults to 1.

Returns:

A tensor containing the symmetric mean absolute percentage error.

Return type:

torch.tensor
